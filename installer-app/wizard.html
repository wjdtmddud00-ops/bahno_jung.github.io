시작 경로 사이트 이름 필수 프로그램 GitHub 연동 설치

시작

디자인과 편의 기능이 포함된 사진 사이트 템플릿을 설치합니다.
설치 경로와 사이트 이름을 설정한 뒤 설치를 진행합니다.

설치 경로

* 경로를 선택해야 다음 단계로 진행할 수 있습니다.

사이트 이름

* 사이트 이름(로고 상단)을 입력해야 다음 버튼을 누를 수 있습니다.

필수 프로그램

아래 프로그램이 없으면 설치를 선택하세요. 이미 최신 버전이 있다면 설치할 필요 없습니다.

Node.js를 설치하시겠습니까?

테스트 서버 실행 및 npm 사용에 필요합니다.

Git을 설치하시겠습니까?

GitHub 페이지에 푸시(배포)할 때 필요합니다.

GitHub 연동

GitHub에 로그인한 뒤, 새 저장소를 만들고 사이트 퍼블리싱 설정까지 설치 중에 자동으로 진행합니다.

토큰 발급 버튼으로 GitHub에서 새 토큰을 만든 뒤 붙여넣으세요. (repo 권한 체크)

사이트 주소

권장: username.github.io 로 두면 https://사용자명.github.io 로 열립니다.

설치 시 GitHub에 새 저장소 생성 → 원격 연결 → 첫 푸시 → GitHub Pages 설정까지 자동으로 진행됩니다. 이후에는 콜렉션·사진만 추가하고 푸시 실행 파일만 누르면 됩니다.
